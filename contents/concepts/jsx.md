# JSX

If you know HTML, you know JSX.

This is valid JSX:

```
<h1>Hello, World!</h1>
```

This is also valid JSX:

```
var name = "Penny";

<h1>Hello, {name}!</h1>
```

JSX is XML-like syntax and any JavaScript enclosed in curly braces, `{}`, will be evaluated. So the above statement will evaluate to `Hello, Penny!`.
